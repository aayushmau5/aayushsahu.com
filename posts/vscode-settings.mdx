# Visual styling

Let's first talk about theming. I think its the most popular topic of all, since all the tech blogs, or videos I see on the internet is filled with comments like "what's that theme?".

Here's how my vscode looks like(as of July 2022):

![My VsCode Setup](/blogImages/vscode.png)

Alright! Let's see, the theme is [Vitesse Dark theme](https://github.com/antfu/vscode-theme-vitesse). I use [JuliaMono](https://juliamono.netlify.app/) as my editor font, and my terminal font is [Fira Code Nerd Font](https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/FiraCode). For Icons, I use [File Icons](https://marketplace.visualstudio.com/items?itemName=file-icons.file-icons)(you can change your file icons using command palette(`Ctrl-p`), and search for "File Icon theme"). For Product icon(the icons that you see on the right most sidebar), I use [Carbon Product Icons](https://marketplace.visualstudio.com/items?itemName=antfu.icons-carbon)(you can change this using command palette(`Ctrl-p`), and searching for "Product Icon theme").

## Font configurations

```json
{
  "editor.fontLigatures": true,
  "editor.fontFamily": "'JuliaMono'",
  "editor.fontWeight": "400",
  "editor.letterSpacing": 0.1,
  "editor.lineHeight": 25,
  "terminal.integrated.fontSize": 15,
  "terminal.integrated.fontWeightBold": "500",
  "terminal.integrated.fontFamily": "'FuraCode Nerd Font'"
}
```

All the editor related settings have `editor` prefix. To get all the lists of configurations you can make, just enter `editor.` and VSCode will give you all the hints ;)

Most of the stuff is pretty straightforward. You can change the font weight, spacing between letters, line height, etc.

To edit terminal configurations, you can use `terminal` prefix. Here, I changed my integrated terminal's font, it's weight and size. That's it for font configurations, lets move on to configuring some colors.

## Color configurations

```json
{
  "editor.rulers": [
    {
      "column": 80,
      "color": "#4f545a40"
    }
  ],
  "workbench.colorCustomizations": {
    "[Default Dark+]": {
      "statusBar.background": "#1b1b1b",
      "activityBar.background": "#1b1b1b",
      "sideBar.background": "#1b1b1b",
      "editor.background": "#1b1b1b"
    }
  },
  "editor.tokenColorCustomizations": {
    "textMateRules": [
      {
        "scope": ["comment"],
        "settings": {
          "fontStyle": "italic"
        }
      }
    ]
  }
}
```

Let's start with `editor.rulers`, with this you can put a vertical line in your editor. I have set it to 80 columns. I don't strictly abide by those 80 column width, but it helps as a visual inidication whether I should break-up the lines or not.

With `workbench.colorCustomizations`, you can customize existing themes. Say if you like your existing theme but wanna make slight modifications to it, you can do it this way. Just provide the name of the theme you want to config(vscode provides hint for this as well), and make the changes you want.

With `editor.tokenColorCustomizations`, you can customize what each token looks like. Here I'm configuring that all comments have `italic` style applied to them.

    "key": "ctrl+t",
    "command": "workbench.action.terminal.toggleTerminal"
  },
  {
    "key": "ctrl+cmd+enter",
    "command": "workbench.action.toggleMaximizedPanel"
  },
  {
    "key": "cmd+d",
    "command": "workbench.action.terminal.split",
    "when": "terminalFocus && terminalProcessSupported"

  {
    "key": "ctrl+k",
    "command": "workbench.action.terminal.focusNextPane",
    "when": "terminalFocus && terminalProcessSupported"
  },
  {
    "key": "ctrl+j",
